# CARA MENJALANKAN BACKEND
## 1. Atur File app.py
1. Buka file app.py
2. Masukkan _Hugging Face Token_, _Database URL_, dan _Gemini Api Key_ milik anda ditempat yang telah disediakan

## 2. Menjalankan backend
1. Buat virtual environment menggunakan python ( py -m venv env )
2. Aktifkan virtual environment ( source env/Scripts/activate [jika menggunakan git bash] )
3. Install semua yang ada di requirements.txt ( py -m pip install -r requirements.txt )
4. Jalankan backend ( python app.py )
